## ğŸ†˜ NEED HELP?

### Common Issues & Solutions

**Problem:** `hugo: command not found`
**Solution:**
```bash
# Install Hugo
brew install hugo  # macOS
choco install hugo-extended  # Windows
snap install hugo  # Linux
```

**Problem:** CSS not loading
**Solution:**
```bash
hugo --gc  # Clear cache
hugo server -D  # Restart server
```

**Problem:** CMS not loading
**Solution:**
- Check internet connection (CMS loads from CDN)
- Verify files exist: `/admin/index.html` and `/admin/config.yml`
- Clear browser cache (Ctrl+Shift+R)

**Problem:** Can't login to CMS
**Solution:**
- Local: CMS shows interface but can't save (normal)
- Live: Ensure Git Gateway is enabled in Netlify
- Check Identity service is active
